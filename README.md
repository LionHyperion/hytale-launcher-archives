# Hytale Launcher Archives

This repository contains archived builds of the Hytale Launcher for historical preservation.

## Builds

### Version 1.0
- Initial release version of the Hytale Launcher

### release2026.01.10-5fdaa5c
- Build date: January 10, 2026
- Commit hash: 5fdaa5c
- Latest release build

## Contents

Each build folder contains:
- `hytale-launcher.exe` - The main launcher executable
- `uninstall.exe` - Uninstaller (if available)
- `eula.txt` - End User License Agreement
- `third-party-licenses.txt` - Third-party software licenses
- `.keys/` - License keys and authentication files
- `install/` - Installation files and packages

## Notes

- Cache files, logs, and temporary WebView2 data are excluded via `.gitignore`
- Installer executables are excluded to reduce repository size
- Only essential launcher files and documentation are preserved

## Purpose

This archive serves as a historical record of the Hytale Launcher's development and evolution over time.
